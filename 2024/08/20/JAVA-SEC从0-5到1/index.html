从0.5开始的java安全

java安全的门槛太高了….尤其是面对Java语言的抽象性时

早在2024年初，我就萌生了学习Java安全的念头，但Java的复杂性让我望而却步。之后在参加xyctf时，N1ght师傅推荐了几篇关于Java安全的文章，可是那时连基础语法我都不太掌握，学习了两天就放弃了。

后面丢丢捡捡总算是把java的基础学完了，后面java EE的知识更是痛苦至极，光是配置一些简单的环境就能耗费我一天的时间，学了一个星期也算是把最基础的一些学完了，但是也没有很扎实，反射 动态代理 类加载这种在java安全里很重要的知识也是因为天赋太笨没学会。后面简单的看了几条链子，跟着跟了也没跟会。

八月一号入职安恒之后，加剧了我想要掌握Java安全的决心，打算从这里开始把java安全学好

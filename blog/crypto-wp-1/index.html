from random import choice
from Crypto.Util.number import isPrime, sieve_base as primes
from flag import flag

def getPrime(bits):
while True:
  n = 2
  while n.bit_length() < bits:
      n *= choice(primes)	#primes为前10000个素数的列表
  if isPrime(n + 1):
	  return n + 1

e = 0x10001
m = int.from_bytes(flag.encode(), 'big')
p, q = [getPrime(2048) for _ in range(2)]
n = p * q
c = pow(m, e, n)

已知n,c,e，同时观察到p,q构造比较特殊是多个素数相乘+1构造得到的。

我们直接先求p p按照上面等同于

p=p1a1×p2a2×......×pnan+1p=p_1^{a_1}\times p_2^{a_2}\times ......\times p_n^{a_n}+1p=p1a1​​×p2a2​​×......×pnan​​+1

根据费马小定理

ap−1≡1(modp)a^{p-1}\equiv 1 \pmod{p}ap−1≡1(modp)

假设a=2（a可以是任意数，为了方便一般是2） 上式子可转变成

2k⋅(p−1)≡1(modp)2^{k·(p-1)}\equiv 1 \pmod{p}2k⋅(p−1)≡1(modp)

那么

p∣2k⋅(p−1)−1p\mid 2^{k·(p-1)}-1p∣2k⋅(p−1)−1

同时

p∣np\mid np∣n

所以

n∣2k⋅(p−1)−1n\mid 2^{k·(p-1)}-1n∣2k⋅(p−1)−1

那么我们就可以通过找到两数最大公因数得到p，那么问题是我们要怎么构造k⋅(p−1)k·(p-1)k⋅(p−1)，我们发现p是由很多质数相乘得到，那么我们就可以把2~最大质数的结果相乘，一定是和p整除的，最终解题代码如下：
